# YggdrasilHost Installation Script

This repository contains the automated installation script for YggdrasilHost - a bare metal Arch Linux server with Hyprland desktop environment. The script takes you from a fresh Arch ISO boot to a fully working system with NVIDIA RTX 2070 Super graphics, Docker-ready configuration, and essential services. It handles disk partitioning, driver installation, desktop environment setup, and user configuration automatically with minimal user interaction required.
